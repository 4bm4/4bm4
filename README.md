## About me

I'm an ophthalmologist working in Madrid, with a particular interest in glaucoma, cataract surgery, refractive surgery and clinical imaging.

Alongside my clinical work, I spend a good part of my time exploring how programming, data science and machine learning can be used in real ophthalmology problems. I am especially interested in reproducible image analysis, small clinical datasets and tools that can help turn diagnostic imaging into measurable biomarkers.

At the moment, my research is focused mainly on corneal phenotyping, keratoconus, anterior segment imaging and clinical machine learning pipelines.

I use this GitHub profile to share projects, experiments and code related to:

- ophthalmology and clinical research
- medical image analysis
- corneal topography and tomography
- machine learning applied to small clinical cohorts
- personal tools, prototypes and web projects

## Current work

Some of the things I am currently working on:

- **Corneal phenotyping and keratoconus**  
  Analysis of corneal topography and tomography maps, with a focus on reproducible descriptors and clinically meaningful patterns.

- **Clinical image analysis pipelines**  
  Tools for anonymized image processing, feature extraction, validation and visualization in ophthalmology datasets.

- **Personal medical website**  
  [drfranciscoabellan.com](https://drfranciscoabellan.com/), built with React, TypeScript, Vite and Tailwind CSS.
